3 Requisitos

  • Euro Truck Simulator 2 RECOMENDADO ou American Truck Simulator
  • Plugin SDK habilitado nas configurações do jogo
  • Conexão com internet para enviar os dados ao ranking
  • Windows 10/11 (64 bits)

4 Como funciona

  • O app instala automaticamente o plugin de telemetria no ETS2/ATS
  • Durante o jogo, as viagens são detectadas e registradas automaticamente
  • Os dados vão para o servidor central e entram no ranking
  • Você acompanha suas estatísticas, conquistas e prêmios pelo site
