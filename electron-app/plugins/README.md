# Plugins do Cargo Stats

Coloque aqui os arquivos do servidor de telemetria:

- `ets2-telemetry-server.exe` — Servidor HTTP de telemetria (porta 25555)
- `ets2-telemetry-server.dll` — Plugin do jogo (instalado automaticamente no ETS2/ATS)
- `plugins/` — Pasta com as DLLs dos jogos, se houver

## Onde baixar

https://github.com/FurballFriendship/ets2-telemetry-server

Baixe a versão mais recente e extraia os arquivos nesta pasta.

O instalador do Cargo Stats vai:
1. Copiar a `.dll` para a pasta de plugins do ETS2/ATS
2. Iniciar o servidor de telemetria automaticamente
